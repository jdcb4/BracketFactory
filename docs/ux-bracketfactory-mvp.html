BracketFactory MVP Wizard Outline

The MVP starts with one bracket type: a flat L bracket. The angle bracket remains visible as a disabled future option so the product direction is clear without splitting the constraint model too early. The working layout is a fixed-height wizard: the linked category ribbon stays across the top, the active parameters stay in the left column, and the compact summary plus interactive preview stay stacked on the right.

1. Type

Choose Flat L Bracket. Angle Bracket is shown as coming next.

2. Size

Set leg lengths, leg width, and thickness. These values constrain every later step.

3. Hardware

Select metric or screw hardware from a compact control, then choose through or countersunk holes. The selected head diameter determines safe edge distance and pitch when countersinking is enabled.

4. Holes

Pick hole counts and rows within computed safe limits. Invalid counts are clamped.

5. Export

Review derived dimensions and download STL or 3MF generated in the browser.

Constraint posture

  • Hole centers stay away from edges using the larger of clearance-hole and countersink-head protection.
  • Hole pitch prevents both straight holes and countersink seats from overlapping.
  • Hole rows are limited by leg width and hardware clearance.
  • Inside-corner holes are excluded from the shared corner square.
  • The preview is an interactive Three.js mesh generated from the same JSCAD solid as export.
